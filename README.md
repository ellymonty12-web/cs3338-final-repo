# CS 3338 Final Project

## Disclaimer

This repository is for a practice project only and serves as a simulation, not an actual working project. It is not affiliated with, endorsed or sponsored by, or connected to Want2Remember or We2Link. There is no connection to the original senior design team or any other related organization.

## Project Overview

This is mock software development project inspired by Want2Remember on the ascent web site. This would web app that helps users with memory problems store important information, reminders, contacts, appointments, tasks, medication information, and other daily living information in one place. The repository is used to simulate the devolpment process, documentation, project management, testing plans, use of Docker, diagrams and snapshots which would be used in such a project.

## Why this software matters

People with memory difficulties may need help remembering important information such as appointments, medical information, contacts, and tasks. This project simulates an app that would organize that information in one convenient place.
