# Final Project

## Disclaimer

This repository is only a practice simulation for a final project. It is based on the senior design project Want2Remember. This repository is not affiliated with Want2Remember, We2Link, the original project, the original design team, or any related organization. No actual application is being developed or distributed here.

## Project Overview

This project is simulated as a memory-support application for users who need help organizing memories, contacts, appointments, reminders, and daily tasks. This repository contains documentation, planning artifacts, test reports, Docker service planning, and GitHub snapshots.
