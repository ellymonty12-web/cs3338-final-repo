\section{Introduction}
\subsection{Purpose}
This document describes the simulated requirements for an application based on Want2Remember from the ascent website.

\subsection{Product Scope}
The simulated system helps users create, organize, search, and view memories, reminders, tasks, and contacts.

\section{External Interface Requirements}
\subsection{User Interface}
The system shall include a home page, login page, memory page, create memory page, contacts page, and search page.

\subsection{Software Interfaces}
The simulated system shall use React, React Native, Firebase Authentication, Firestore-style storage, Jira, TestRail, Docker Compose, and GitHub.

\section{Functional Requirements}

FR-4 & The system shall allow users to view contacts. \\ \hline

\end{longtable}

\section{Legal and Ethical Considerations}
The simulated system may involve sensitive personal information. The design should consider privacy, consent, secure storage, and responsible handling of user data.
