\section{Snapshot 2 Design Update: Memory Creation Feature}

Snapshot 2 adds the simulated design for memory creation and memory templates.

\subsection{Create Memory Page}

The Create Memory page allows the user to begin creating a new memory entry. The page would include a title field, category selection, date field, notes field, and save button.

\subsection{Template Selection Page}

The Template Selection page allows the user to choose what type of memory they want to create. Planned templates include appointment, medication, task, contact-related memory, password entry, and custom note.
